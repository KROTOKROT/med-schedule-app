# 11. User Guide (Руководство пользователя)

## 1. Общая информация

Данное руководство описывает использование мобильного приложения MedSchedule.

Приложение предназначено для управления расписанием приёма лекарств, создания напоминаний и просмотра истории приёмов.

---

## 2. Начало работы

### 2.1 Установка приложения

- Установить APK (Android)
- Или запустить через Android Studio / Flutter run

---

### 2.2 Первый запуск

При первом запуске пользователь:
- регистрируется в системе
- или выполняет вход
- получает доступ к основному функционалу

---

## 3. Авторизация

### Регистрация

1. Ввести email
2. Ввести пароль
3. Нажать "Register"

### Вход

1. Ввести email
2. Ввести пароль
3. Нажать "Login"

После входа выдаётся JWT-токен автоматически.

---

## 4. Основные функции

---

### 4.1 Главный экран

Пользователь видит:
- ближайшие приёмы лекарств
- статус уведомлений
- быстрый доступ к добавлению

---

### 4.2 Добавление лекарства

1. Открыть экран "Add Medication"
2. Ввести название лекарства
3. Указать дозировку
4. Выбрать время приёма
5. Нажать "Save"

---

### 4.3 Список лекарств

Пользователь может:
- просматривать все лекарства
- открывать детали
- удалять записи

---

### 4.4 Детали лекарства

Доступно:
- просмотр информации
- редактирование данных
- удаление

---

### 4.5 Настройки

Пользователь может:
- включать/выключать уведомления
- управлять синхронизацией
- просматривать информацию о приложении

---

## 5. Уведомления

Приложение автоматически:
- отправляет напоминания о приёме лекарств
- работает в фоновом режиме
- поддерживает push/local notifications

---

## 6. Оффлайн режим

При отсутствии интернета:
- данные сохраняются локально
- используется кеш (Room/SQLite)
- синхронизация выполняется при восстановлении сети

---

## 7. Возможные ошибки

- ошибка авторизации → проверить логин/пароль  
- нет данных → проверить подключение к серверу  
- уведомления не приходят → проверить разрешения  

---

## 8. Безопасность

- данные защищены JWT
- пароли хранятся в зашифрованном виде (BCrypt)
- доступ к API ограничен ролями

---

## 9. Советы пользователю

- регулярно обновляйте список лекарств
- включайте уведомления
- проверяйте расписание ежедневно

---

## 10. Вывод

Приложение MedSchedule обеспечивает удобное управление приёмом лекарств, поддерживает оффлайн режим и уведомления, что повышает надёжность соблюдения расписания лечения.
